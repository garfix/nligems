Welcome to the website

Hi, my name is Patrick van Bergen.

On this website I examine and honour the brave attempts to make Natural Language Interfaces. It is a personal website and all views are based on my accumulating, but ever incomplete knowledge of the subject.

The website is a way to examine historic NLI systems in a systematic way. By studying it you can learn more about the structure and common features of NLI.

I hope it will interest you.

What systems are included?

I just include the systems that make a true effort to allow the user to approach a database with the full range of natural language.

Further restrictions I make:

  • Allow users to enter a complex natural language query
  • Attempt to find the answer in one or more structured databases (no free text search)
  • Use a rule-based, exact, approach to find the answer
  • Respond to the question in natural language
  • The data and rules are extendible by non-programmers
  • It has a public, detailed, description of its workings

I might even include a system that fulfills only a single one of these requirements, but it would have to be a historically significant contribution.

When I was creating this website I found much inspiration from the the following overview article. I may quote from it from time to time.

Natural Language Interfaces to Databases - An Introduction (1995): I. Androutsopoulos, G.D. Ritchie, P. Thanisch
